Get Started with Snuushco

Follow these simple steps to begin your journey with Snuushco's powerful software solutions.

1

Create Your Account

Sign up for a free account to access our platform and start building.

curl -X POST https://api.snuushco.com/v1/auth/register \ -H "Content-Type: application/json" \ -d '{"email": "your@email.com", "password": "your-password"}'
2

Install Our SDK

Add our SDK to your project using your preferred package manager.

npm install @snuushco/sdk # or pip install snuushco-sdk
3

Initialize Your Project

Configure your project with your API credentials.

import { Snuushco } from '@snuushco/sdk'; const snuushco = new Snuushco({ apiKey: 'your-api-key', environment: 'development' });
4

Start Building

Begin integrating our modules and automation tools into your workflow.

// Example: Create an automation workflow const workflow = await snuushco.workflows.create({ name: 'My First Workflow', triggers: ['webhook'], actions: ['transform', 'notify'] });
