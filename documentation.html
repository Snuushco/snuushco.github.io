Getting Started

Welcome to the Snuushco documentation. This guide will help you get started with our platform and APIs.

Prerequisites

  • Node.js 14+ or Python 3.8+
  • A Snuushco account
  • API credentials (available in your dashboard)

Installation

Install the Snuushco SDK using your preferred package manager:

# Using npm npm install @snuushco/sdk # Using pip pip install snuushco-sdk

Authentication

All API requests require authentication using your API key:

const snuushco = new Snuushco({ apiKey: 'your-api-key', environment: 'development' });

API Reference

GET /api/v1/workflows

List all workflows in your account.

POST /api/v1/workflows

Create a new workflow.

PUT /api/v1/workflows/{id}

Update an existing workflow.

DELETE /api/v1/workflows/{id}

Delete a workflow.

SDKs

We provide official SDKs for popular programming languages:

  • JavaScript/TypeScript
  • Python
  • Java
  • Go

Webhooks

Configure webhooks to receive real-time updates about your workflows and automations.

// Example webhook payload { "event": "workflow.completed", "data": { "workflow_id": "wf_123", "status": "success", "timestamp": "2024-03-20T10:00:00Z" } }

Examples

Here are some common use cases and examples:

// Create a new workflow const workflow = await snuushco.workflows.create({ name: "My First Workflow", triggers: ["webhook"], actions: ["transform", "notify"] }); // Add a step to the workflow await snuushco.workflows.addStep(workflow.id, { type: "transform", config: { input: "{{trigger.data}}", template: "Hello, {{name}}!" } });

Troubleshooting

Common issues and their solutions:

API Rate Limits

If you're hitting rate limits, consider implementing exponential backoff in your requests.

Authentication Errors

Ensure your API key is valid and has the necessary permissions.
